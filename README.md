### Notes
* Math.floor()

		Math.floor() function returns the largest integer less than or equal to a given number:
			console.log(Math.floor(5.95));
			// expected output: 5

			console.log(Math.floor(5.05));
			// expected output: 5

			console.log(Math.floor(5));
			// expected output: 5

			console.log(Math.floor(-5.05));
			// expected output: -6
